# Hikari 🏯

A landing page showcasing the top places in Japan.

## Features

- **Interactive Map**: Leaflet-powered map to explore Japanese destinations
- **Image Gallery**: Carousel gallery showcasing beautiful locations across Japan
- **Responsive Design**: Fully responsive layout built with Bootstrap 5
- **Travel Tips**: Essential guides for language, transportation, and best times to visit
- **Smooth Navigation**: Fixed navigation with smooth scrolling to sections
- **Loading Animation**: Custom loading screen for better UX

## Tech Stack

- HTML5
- CSS3 (Custom + Bootstrap 5)
- Vanilla JavaScript
- **Libraries**:
  - Bootstrap 5.3.3
  - Leaflet 1.9.4 (Interactive Maps)
  - Font Awesome 6.5.1 (Icons)
  - Google Fonts (Alegreya Sans SC)

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/c-ent/Hikari.git
cd Hikari
```

2. Open `index.html` in your browser:
```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# On Windows
start index.html
```

## Customization

- Edit CSS files in `css/` for styling changes
- Modify `js/script.js` for functionality updates
- Add or replace images in the `images/` folder

## Features Overview

- **Home Section**: Hero carousel featuring Tokyo, Kyoto, and Osaka
- **Discover**: Introduction to Japanese culture and seasons
- **Gallery**: Interactive image carousel with 10 destination photos
- **Explore Map**: Interactive Leaflet map of Japan with key destinations
- **Travel Tips**: Cards covering language, transportation, and timing
