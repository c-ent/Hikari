# Hikari

## Landing Page

Hikari is a landing page based on Japan, designed to show the imagery, travel tips, and highlights. Built with HTML, CSS (including Bootstrap), and JavaScript.
---

---

### Usage
1. Clone the repository:
	```sh
	git clone https://github.com/c-ent/Hikari.git
	```
2. Open `index.html` in your browser.
3. Customize styles or content as needed in the `css/`, `images/`, and `js/` folders.

---

### Technologies Used
- HTML5
- CSS3 & SCSS
- Bootstrap 5
- JavaScript
